- 登录页面，本来是用了bing的每日图片的接口的，但是被说丑，就删了。。。

  ![image-20220107111819014](https://gitee.com/xwyzsn/Picture/raw/master/image-20220107111819014.png)

  - 第一块：根据用户访问的IP地址，使用天气接口，提示天气数据
